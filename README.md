* **Zero-Touch Provisioning**: Configures Java, Jenkins, Docker, and Security settings without a single manual SSH command.

* **Role-Based Architecture**: Modular design separating Jenkins logic from Docker logic for maximum reusability.

### **Node 1: Jenkins CI Master**

* Cleans up legacy containers to ensure a fresh deployment environment.
